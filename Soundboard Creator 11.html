Soundboard Tutorial

  • Add sounds: Select a button, then upload an audio file and click "Add to Selected"
  • Play sounds: Click on any button with an assigned sound
  • Edit sounds: Click the pencil icon on a sound button to edit its properties
  • Adjust volume: Use the slider at the bottom of each button for individual volume
  • Keyboard shortcuts: Use SPACE, ESC, and number keys for quick control
  • Save/Load: Save your layout to JSON and load it later
